## Overview

ZoomTots is an experimental AI-assisted educational platform designed to make learning easier for primary school students.
The system converts uploaded presentation (PPT) study materials into simplified teaching videos where a virtual character explains the topic in a clear and engaging manner.

The goal of the project is to explore how artificial intelligence tools and automated processing can be used to transform static learning content into interactive teaching experiences.



## Problem Statement

Most students struggle to understand textbook explanations or presentation slides without a teacherâ€™s guidance.
Especially for younger learners, PPT slides alone are not engaging and often difficult to follow.

Teachers are not always available for repeated explanations.
Therefore, there is a need for a system that can automatically interpret study material and present it in a simpler, teacher-like format.

ZoomTots attempts to solve this by automatically converting study slides into explainable video lessons.

## System Workflow

1. User uploads a presentation file (PPT).
2. The system extracts slide text content.
3. Content is reorganized into explainable teaching steps.
4. Narration is generated using automated text-to-speech.
5. Slides and narration are synchronized.
6. An animated character presents the topic as a teacher.
7. Student watches the generated lesson.
8. An MCQ quiz evaluates understanding.
